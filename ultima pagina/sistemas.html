Descripción de la Carrera

La carrera de Técnico en Sistemas Informáticos está orientada a la formación de estudiantes con competencias en el manejo de software, hardware y redes, capaces de diseñar, desarrollar y mantener sistemas tecnológicos en distintos entornos laborales.

Formación y Material de Trabajo

Durante los tres años de estudio, los estudiantes adquieren conocimientos sólidos en:

  • Programación

    Aprendizaje de lenguajes como HTML, CSS, JavaScript, Python y Java para crear aplicaciones web y de escritorio.

  • Redes y Hardware

    Configuración de redes locales, mantenimiento de equipos y montaje de sistemas informáticos.

  • Bases de Datos

    Gestión y administración de datos utilizando MySQL, PostgreSQL y herramientas de análisis.

  • Herramientas Digitales

    Uso de software profesional, entornos de desarrollo (VS Code, NetBeans) y control de versiones con GitHub.

Proyectos y Emprendimientos

Los estudiantes desarrollan proyectos tecnológicos que fortalecen su experiencia práctica y fomentan el emprendimiento digital.

Desarrollo Web Escolar

Creación de sitios web institucionales y plataformas educativas con HTML, CSS, JavaScript y PHP.

Aplicaciones Móviles

Diseño de apps para Android usando Java y Kotlin, orientadas a la gestión educativa y comunicación.

Emprendimientos Tecnológicos

Creación de microempresas de soporte técnico, ciberseguridad, diseño web y mantenimiento de redes.
